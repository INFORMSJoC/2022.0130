For solving the CNP problem, under a Windows OS system, please run the executable pragram 'IRMS.exe' with following five parameters
InstanceName is the name of each instance
Dataset is the type of the tested instance
TimeLimit is the time limit (in seconds) for each run of program
NbrRepats is the number of times of running the program
————————————————————————————————————————
./IRMS.exe  InstanceName  Dataset  TimeLimit  NbrRepeats
————————————————————————————————————————
