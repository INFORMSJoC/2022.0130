                             For Solving CNP Instances
*****************************************************************************************************************************************************
Under a Windows OS system, please run the executable pragram 'IRMS.exe' with following five parameters
InstanceName is the name of each instance, e.g., Bovine.txt
Dataset is the type of the tested instance, e.g., realworld
TimeLimit is the time limit (in seconds) for each run of program, e.g., 3600 seconds
NbrRepats is the number of times of running the program, e.g., 20
————————————————————————————————————————
./IRMS.exe  InstanceName  Dataset  TimeLimit  NbrRepeats
————————————————————————————————————————
